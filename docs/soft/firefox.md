# Firefox

Firefox — свободный веб-браузер с открытым исходным кодом, разработанный Mozilla Foundation и Mozilla Corporation на основе движка Quantum. Один из старейших и наиболее популярных браузеров, первый выпуск которого состоялся в 2002 году. Firefox известен своей приверженностью принципам открытого интернета, защитой приватности пользователей и богатыми возможностями настройки через систему расширений.

## Издания Firefox

**Firefox Release** — стабильная версия с регулярными обновлениями каждые 4 недели. Рекомендуется для повседневного использования.

**Firefox ESR (Extended Support Release)** — версия с длительным сроком поддержки, обновляемая раз в год с точечными обновлениями безопасности. Предназначена для организаций и пользователей, которым нужна максимальная стабильность.

**Firefox Beta** — тестовая версия с новыми функциями, выходящая за 4 недели до релиза стабильной версии.

**Firefox Nightly** — ночные сборки с самыми последними изменениями, обновляются дважды в день. Для тестировщиков и энтузиастов.

**Firefox Developer Edition** — специальная версия для веб-разработчиков с дополнительными инструментами отладки.

## Основные возможности

**Усиленная защита от отслеживания** — встроенная блокировка трекеров, блокировщик cookies сторонних сайтов и защита от цифровых отпечатков браузера.

**Открытый исходный код** — полностью открытый код под лицензией MPLv2.0, позволяющий сообществу проверять безопасность и вносить улучшения.

**Мощная система расширений** — тысячи дополнений для расширения функциональности браузера через WebExtensions API.

**Синхронизация Firefox** — безопасная синхронизация закладок, паролей, истории и открытых вкладок между всеми устройствами.

**Контейнеры вкладок** — изоляция cookies и данных сайтов в разных контейнерах для повышения приватности.

**Приватный просмотр** — режим, не сохраняющий историю, cookies и временные файлы.

## Установка

Выберите подходящий способ установки Firefox для вашей системы:

::: code-group

```shell [UBUNTU, DEBIAN, MINT, POPOS]
# Создание директории для ключей
sudo install -d -m 0755 /etc/apt/keyrings

# Импорт GPG ключа Mozilla
wget -q https://packages.mozilla.org/apt/repo-signing-key.gpg -O- | sudo tee /etc/apt/keyrings/packages.mozilla.org.asc > /dev/null

# Добавление репозитория Mozilla
echo "deb [signed-by=/etc/apt/keyrings/packages.mozilla.org.asc] https://packages.mozilla.org/apt mozilla main" | sudo tee -a /etc/apt/sources.list.d/mozilla.list > /dev/null

# Установка приоритета репозитория
echo '
Package: *
Pin: origin packages.mozilla.org
Pin-Priority: 1000
' | sudo tee /etc/apt/preferences.d/mozilla

# Обновление и установка
sudo apt update
sudo apt install firefox
```

```shell [FEDORA]
sudo dnf install firefox
```

```shell [OPENSUSE]
sudo zypper install MozillaFirefox
```

```shell [ARCH LINUX]
sudo pacman -S firefox
```

```shell [ALT LINUX]
# Обновление и установка
su -
apt-get update
apt-get dist-upgrade
apt-get install firefox
```

```shell [REDOS]
su -
dnf install firefox
```

```shell [FLATPAK]
flatpak install flathub org.mozilla.firefox
```

```shell [SNAP]
sudo snap install firefox
```

```shell [EPM]
epm install firefox
```

:::

::: info
В большинстве дистрибутивов Linux Firefox предустановлен и является браузером по умолчанию. Установка через официальный репозиторий Mozilla обеспечивает получение самых свежих обновлений.
:::

::: warning
При установке Snap или Flatpak версии Firefox может работать в изолированной среде с ограниченным доступом к системным файлам. Для полной интеграции рекомендуется использовать нативные пакеты.
:::

## Особенности работы

### Усиленная защита от отслеживания

Firefox предлагает три уровня защиты:

**Стандартный** — блокирует известные трекеры в приватных окнах, социальные трекеры во всех окнах и cookies для отслеживания между сайтами.

**Строгий** — блокирует все известные трекеры, социальные трекеры, cookies для отслеживания, криптомайнеры и цифровые отпечатки браузера.

**Персональный** — позволяет вручную настроить параметры блокировки для максимального контроля.

Для настройки:
1. Откройте Настройки → Приватность и защита
2. Выберите уровень защиты в разделе «Усиленная защита от отслеживания»
3. Включите «Отправлять сайтам сигнал "Не отслеживать"»

### Контейнеры вкладок

Контейнеры позволяют изолировать сайты друг от друга:

1. Установите расширение [Firefox Multi-Account Containers](https://addons.mozilla.org/firefox/addon/multi-account-containers/)
2. Создайте контейнеры для разных целей (Работа, Личное, Покупки)
3. Открывайте вкладки в нужных контейнерах через контекстное меню
4. Настройте автоматическое открытие сайтов в определённых контейнерах

::: tip
Контейнеры полезны для одновременной работы с несколькими аккаунтами одного сервиса или для изоляции потенциально небезопасных сайтов.
:::

### Синхронизация Firefox Sync

Настройка синхронизации данных между устройствами:

1. Откройте Настройки → Firefox Аккаунт
2. Создайте аккаунт или войдите в существующий
3. Выберите данные для синхронизации:
   - Открытые вкладки
   - Закладки
   - Пароли
   - История
   - Дополнения
   - Настройки
4. Повторите процесс на других устройствах

::: info
Все синхронизированные данные шифруются на устройстве перед отправкой и могут быть расшифрованы только с вашим паролем.
:::

## Полезные советы и дополнения

### Настройка внешнего вида

**Темы оформления** — выбирайте между светлой, тёмной и системной темой:
- Настройки → Расширения и темы → Темы

**Панель инструментов** — настройте расположение кнопок:
1. Нажмите правой кнопкой на панели инструментов
2. Выберите «Настроить панель инструментов»
3. Перетащите элементы в нужное место

**Плотность интерфейса** — выберите компактный, стандартный или свободный режим:
- Настройки → Расширения и темы → Плотность

### Ускорение работы браузера

Для улучшения производительности:

1. **Аппаратное ускорение** — убедитесь, что включено:
   - Настройки → Основные → Производительность → Использовать рекомендуемые настройки производительности

2. **Ограничение процессов контента**:
   - about:config → `dom.ipc.processCount` → установите значение 4-8

3. **Включение WebRender**:
   - about:config → `gfx.webrender.all` → установите `true`

```sh
# Проверка использования WebRender
about:support
# Найдите раздел "Graphics" и строку "Compositing"
```

### Установка расширений

Firefox поддерживает богатую экосистему расширений:

1. Откройте [Дополнения Firefox](https://addons.mozilla.org/)
2. Найдите нужное расширение
3. Нажмите «Добавить в Firefox»
4. Управляйте расширениями через about:addons

Рекомендуемые расширения:
- **uBlock Origin** — блокировщик рекламы и трекеров
- **Privacy Badger** — автоматическая защита от отслеживания
- **HTTPS Everywhere** — принудительное использование HTTPS
- **Multi-Account Containers** — контейнеры вкладок

### Работа с профилями

Firefox поддерживает несколько профилей:

```sh
# Запуск менеджера профилей
firefox -ProfileManager

# Создание нового профиля
firefox -CreateProfile "ИмяПрофиля"

# Запуск с определённым профилем
firefox -P "ИмяПрофиля"
```

## Решение проблем

### Тиринг (разрывы изображения)

Если наблюдаются разрывы изображения при скроллинге:

1. Введите `about:config` в адресную строку
2. Найдите `layers.acceleration.force-enabled`
3. Установите значение `true`
4. Перезапустите браузер

### Проблемы с отображением эмодзи

Для исправления чёрно-белых эмодзи:

```sh
# Создайте или отредактируйте файл
nano ~/.config/fontconfig/fonts.conf
```

Добавьте следующее содержимое:

```xml
<selectfont>
  <rejectfont>
    <pattern>
      <patelt name="family">
        <string>DejaVu Sans</string>
      </patelt>
    </pattern>
    <pattern>
      <patelt name="family">
        <string>DejaVu Serif</string>
      </patelt>
    </pattern>
    <pattern>
      <patelt name="family">
        <string>DejaVu Sans Mono</string>
      </patelt>
    </pattern>
  </rejectfont>
</selectfont>
```

Затем выполните:
```sh
fc-cache -fv
```

### Не запускается браузер

Если Firefox не запускается:

**Проверка целостности профиля:**
```sh
firefox -safe-mode
```

**Создание нового профиля:**
```sh
firefox -ProfileManager
```

**Сброс Firefox:**
1. Введите `about:support` в адресную строку
2. Нажмите «Обновить Firefox»
3. Подтвердите операцию

### Высокое потребление памяти

Для оптимизации использования памяти:

1. **Уменьшите количество открытых вкладок**
2. **Отключите неиспользуемые расширения**:
   - about:addons → Отключить ненужные расширения
3. **Включите выгрузку неактивных вкладок**:
   - about:config → `browser.tabs.unloadOnLowMemory` → `true`

### Проблемы с видео и аудио

При проблемах с воспроизведением медиаконтента:

::: code-group

```shell [UBUNTU, DEBIAN, MINT, POPOS]
sudo apt install ubuntu-restricted-extras ffmpeg
```

```shell [FEDORA]
sudo dnf install ffmpeg
```

```shell [ARCH LINUX]
sudo pacman -S ffmpeg
```

:::

**Включение DRM:**
1. Настройки → Основные → Цифровое управление правами (DRM)
2. Включите «Воспроизводить DRM-контролируемое содержимое»

### Перезагрузка кэша

Для принудительной перезагрузки страницы:
- **Обычная перезагрузка:** `Ctrl + R`
- **Жёсткая перезагрузка:** `Ctrl + Shift + R`
- **Полная очистка кэша:** `Ctrl + Shift + Del`

## Дополнительные настройки

### Настройка приватности

Для максимальной защиты конфиденциальности:

1. **DNS over HTTPS (DoH)**:
   - Настройки → Основные → Параметры сети → Настройки
   - Включите «Использовать DNS over HTTPS»
   - Выберите провайдера (Cloudflare или NextDNS)

2. **Очистка данных при закрытии**:
   - Настройки → Приватность и защита → История
   - Включите «Удалять куки и данные сайтов при закрытии Firefox»

3. **Блокировка цифровых отпечатков**:
   - Настройки → Приватность и защита → Усиленная защита от отслеживания
   - Выберите режим «Строгий»

### Импорт данных из других браузеров

Firefox позволяет импортировать данные из других браузеров:

1. Меню → Закладки → Показать все закладки
2. Импорт и резервные копии → Импортировать данные из другого браузера
3. Выберите браузер-источник (Chrome, Edge, Opera и т.д.)
4. Отметьте данные для импорта
5. Нажмите «Далее»

### Использование Firefox GNOME Theme

Для интеграции с GNOME установите Firefox GNOME Theme:

```sh
# Клонирование репозитория темы
git clone https://github.com/rafaelmardojai/firefox-gnome-theme.git ~/.mozilla/firefox-gnome-theme/

cd ~/.mozilla/firefox-gnome-theme/

# Запуск установочного скрипта
./scripts/auto-install.sh -f ~/.mozilla/firefox/
```

**Обновление темы:**
```sh
cd ~/.mozilla/firefox-gnome-theme/
git pull
./scripts/auto-install.sh -f ~/.mozilla/firefox/
```

### Изменение направления текста в адресной строке

Если текст в адресной строке выравнивается справа налево:
- Нажмите `Ctrl + Shift + X` для смены направления

### Отображение вкладок в несколько рядов

Для версий Firefox 71 и выше:

1. Создайте файл `userChrome.css` в директории:
   ```sh
   ~/.mozilla/firefox/<profile>.default-default/chrome/
   ```

2. Добавьте CSS-код из репозитория:
   [tabs_multiple_lines_fx71.css](https://github.com/Aris-t2/CustomCSSforFx/blob/master/classic/css/tabs/tabs_multiple_lines_fx71.css)

3. Включите поддержку пользовательских стилей:
   - about:config
   - `toolkit.legacyUserProfileCustomizations.stylesheets` → `true`

## Системные требования

Для оптимальной работы Firefox на Linux необходимо:

- **Процессор:** Pentium 4 или новее с поддержкой SSE2
- **Оперативная память:** минимум 512 МБ (рекомендуется 2 ГБ и более)
- **Место на диске:** не менее 200 МБ свободного пространства
- **Система:** любой современный дистрибутив Linux
- **Библиотеки:** GTK+ 3.14 или выше, GLib 2.42 или выше, libstdc++ 4.8.1 или выше

## Переход с Firefox ESR на обычную версию

Для смены версии без потери данных:

1. **Экспортируйте закладки** (на всякий случай):
   - Закладки → Показать все закладки → Импорт и резервные копии → Экспорт

2. **Удалите Firefox ESR:**
   ```sh
   sudo apt remove firefox-esr
   ```

3. **Установите обычный Firefox** (см. раздел "Установка")

4. **Профиль сохранится автоматически** в `~/.mozilla/firefox/`

::: info
При переходе с ESR на обычную версию все данные профиля сохраняются, так как они хранятся отдельно от исполняемых файлов браузера.
:::
