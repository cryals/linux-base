# Waterfox

Waterfox — независимый веб-браузер с открытым исходным кодом, основанный на Mozilla Firefox и оптимизированный для 64-битных систем. Создан в 2011 году 16-летним студентом Алексом Контосом для развития идеалов свободного интернета. Waterfox стал одним из первых широко распространённых 64-битных браузеров и быстро завоевал лояльную аудиторию пользователей, ценящих баланс между приватностью и удобством использования.

## Основные возможности

**Приватность по умолчанию** — полностью отключена телеметрия и сбор пользовательских данных, обеспечивая высокий уровень конфиденциальности без компромиссов.

**Производительность** — оптимизация для 64-битных процессоров обеспечивает высокую скорость работы и стабильность даже при большом количестве открытых вкладок.

**Поддержка расширений** — совместимость как со старыми (legacy), так и современными расширениями Firefox, Chrome Web Store и Opera Web Store.

**Защита от отслеживания** — встроенная система Enhanced Tracking Protection блокирует скрытые трекеры и защищает от сбора данных на различных сайтах.

**Приватные вкладки** — возможность открывать приватные вкладки внутри обычного окна без необходимости запуска отдельного приватного окна.

**Настраиваемость** — полная кастомизация интерфейса, тем оформления и функциональности браузера под индивидуальные потребности.

## Установка

Выберите подходящий способ установки Waterfox для вашей системы:

::: code-group

```shell [Ubuntu, Debian, Mint, PopOS]
curl -fsSL https://download.opensuse.org/repositories/home:hawkeye116477:waterfox/xUbuntu_24.04/Release.key | sudo gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/home_hawkeye116477_waterfox.gpg > /dev/null

echo 'deb http://download.opensuse.org/repositories/home:/hawkeye116477:/waterfox/xUbuntu_24.04/ /' | sudo tee /etc/apt/sources.list.d/home:hawkeye116477:waterfox.list

sudo apt update
sudo apt install waterfox-g-kpe
```

```shell [Fedora]
sudo nano /etc/yum.repos.d/waterfox.repo

# Добавьте следующий контент:
[Waterfox]
name=Waterfox Repository
baseurl=https://repo.waterfox.net/fedora/$releasever/
enabled=1
gpgcheck=1
gpgkey=https://repo.waterfox.net/key.asc

# Установка
sudo dnf install waterfox
```

```shell [OpenSuse]
sudo zypper addrepo https://download.opensuse.org/repositories/home:/hawkeye116477:/waterfox/openSUSE_Tumbleweed/home:hawkeye116477:waterfox.repo

sudo zypper refresh
sudo zypper install waterfox
```

```shell [Arch Linux]
# Установка AUR helper (yay)
sudo pacman -S --needed base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

# Установка Waterfox
yay -S waterfox-current-bin
```

```shell [Alt Linux]
epm play waterfox
```

```shell [RedOS]
wget https://github.com/WaterfoxCo/Waterfox/releases/download/G6.0.19/waterfox-G6.0.19.en-US.linux-x86_64.tar.bz2

# Установка
sudo dnf install ./waterfox-*.rpm
```

```shell [Flatpak]
flatpak install flathub net.waterfox.waterfox
```

```shell [EPM]
epm install waterfox
```

:::

::: info
Waterfox рекомендуется устанавливать через менеджер пакетов дистрибутива для оптимальной настройки всех необходимых библиотек.
:::

::: warning
Перед установкой убедитесь, что в системе установлены необходимые библиотеки. Их отсутствие может привести к неработоспособности браузера.
:::

## Особенности работы

### Приватность и безопасность

**Отключение телеметрии** — единственные данные, отправляемые браузером — версия ОС и версия браузера для проверки обновлений.

**Enhanced Tracking Protection** — встроенная защита от трекеров предотвращает скрытый сбор данных о ваших привычках просмотра на различных сайтах.

**Приватный просмотр** — режим приватного просмотра не сохраняет историю, куки, данные форм и не оставляет следов после завершения сессии.

**Oblivious DNS** — затрудняет отслеживание вашим интернет-провайдером подключений к веб-сайтам, сохраняя конфиденциальность онлайн-активности.

### Производительность и совместимость

**64-битная оптимизация** — браузер изначально разрабатывался для 64-битных систем, обеспечивая превосходную производительность и совместимость с современными технологиями.

**Поддержка плагинов** — разрешены все 64-битные NPAPI плагины, включая Java и Silverlight, без «белого» списка ограничений.

**Стабильное API** — плагины и расширения продолжают работать при обновлении браузера благодаря стабильному API.

**Менеджер паролей** — встроенный безопасный менеджер для хранения и управления паролями.

### Синхронизация данных

Waterfox поддерживает синхронизацию через аккаунт Firefox:

1. Откройте Настройки → Синхронизация
2. Войдите в аккаунт Firefox или создайте новый
3. Выберите данные для синхронизации:
   - Закладки
   - История просмотров
   - Открытые вкладки
   - Пароли
   - Дополнения
   - Настройки браузера

::: tip
Синхронизация позволяет продолжать работу на разных устройствах с сохранением всех настроек и данных.
:::

## Полезные советы и дополнения

### Настройка приватности

Для максимальной защиты конфиденциальности:

1. Откройте Настройки → Приватность и защита
2. Настройте уровень защиты от отслеживания:
   - Строгий — максимальная защита
   - Стандартный — баланс между защитой и совместимостью
   - Пользовательский — ручная настройка параметров
3. Включите «Отправлять сайтам сигнал "Не отслеживать"»
4. Настройте очистку данных при закрытии браузера

### Использование приватных вкладок

Waterfox позволяет открывать приватные вкладки внутри обычного окна:

1. Кликните правой кнопкой мыши на панели вкладок
2. Выберите «Новая приватная вкладка»
3. Приватные вкладки помечаются специальным значком
4. Данные приватных вкладок автоматически удаляются при перезапуске

::: info
Приватные вкладки не сохраняют историю, данные форм и предложения автозаполнения, обеспечивая анонимность просмотра.
:::

### Управление расширениями

Waterfox предлагает расширенную поддержку дополнений:

1. Откройте Меню → Дополнения и темы
2. Установите расширения из различных источников:
   - Магазин дополнений Firefox
   - Chrome Web Store (через совместимость)
   - Opera Web Store
3. Управляйте legacy-расширениями через вкладку «Устаревшие расширения»
4. Настройте разрешения для каждого расширения

### Настройка производительности

Для оптимизации скорости работы:

1. Откройте Настройки → Основные → Производительность
2. Включите «Использовать аппаратное ускорение, если доступно»
3. Настройте количество процессов контента (рекомендуется 4-8)
4. Включите функцию гибернации неактивных вкладок

```shell
# Запуск с ограничением процессов
waterfox --process-per-site
```

## Решение проблем

### Не запускается браузер

Если Waterfox не запускается после установки:

**Проверка зависимостей:**

::: code-group

```shell [Ubuntu, Debian, Mint, PopOS]
sudo apt --fix-broken install
```

```shell [Fedora]
sudo dnf install --best --allowerasing
```

:::

**Проверка прав доступа:**
```shell
# Если установлен вручную в /opt
sudo chmod +x /opt/waterfox/waterfox
sudo chown -R root:sudo /opt/waterfox
```

### Проблемы с воспроизведением видео

При проблемах с медиаконтентом:

::: code-group

```shell [Ubuntu, Debian, Mint, PopOS]
sudo apt install ubuntu-restricted-extras ffmpeg
```

```shell [Fedora]
sudo dnf install ffmpeg
```

```shell [Arch Linux]
sudo pacman -S ffmpeg
```

:::

**Настройка кодеков:**
1. Введите `about:config` в адресную строку
2. Найдите `media.ffmpeg.enabled`
3. Установите значение `true`
4. Перезапустите браузер

### Проблемы с расширениями

Если расширение не работает корректно:

1. Проверьте совместимость расширения с текущей версией
2. Откройте Инструменты → Консоль браузера для просмотра ошибок
3. Попробуйте установить альтернативную версию расширения
4. Очистите кэш и перезапустите браузер:
   ```shell
   rm -rf ~/.waterfox/*/cache2/*
   ```

### Сброс настроек браузера

При критических проблемах:

```shell
# Полный сброс настроек
rm -rf ~/.waterfox/

# Сброс только настроек профиля
rm ~/.waterfox/*/prefs.js

# Очистка кэша
rm -rf ~/.waterfox/*/cache2/
rm -rf ~/.waterfox/*/startupCache/
```

::: warning
Сброс настроек удалит все локальные данные. Экспортируйте закладки и пароли перед выполнением этой операции.
:::

## Дополнительные настройки

### Импорт данных из других браузеров

Waterfox позволяет импортировать данные из других браузеров:

1. Откройте Меню → Закладки → Показать все закладки
2. Нажмите «Импорт и резервные копии»
3. Выберите «Импортировать данные из другого браузера»
4. Выберите браузер-источник:
   - Firefox
   - Chrome/Chromium
   - Edge
   - Opera
5. Отметьте данные для импорта и нажмите «Далее»

### Настройка внешнего вида

Персонализация интерфейса браузера:

1. **Темы оформления** — установите готовые темы или создайте собственную
2. **Панель инструментов** — настройте расположение кнопок и элементов управления
3. **Плотность интерфейса** — выберите компактный, стандартный или удобный режим
4. **Шрифты** — настройте размер и тип шрифтов для веб-страниц

Для доступа к настройкам:
- Настройки → Основные → Язык и внешний вид

### Использование поисковых систем

Настройка поисковых систем:

1. Откройте Настройки → Поиск
2. Выберите поисковую систему по умолчанию
3. Добавьте дополнительные поисковые системы
4. Настройте поисковые предложения
5. Используйте ключевые слова для быстрого доступа к поиску

::: tip
По умолчанию Waterfox использует Bing, но вы можете легко изменить это на любую другую поисковую систему.
:::

### Настройки безопасности

Усиление защиты браузера:

1. **HTTPS-Only режим** — принудительное использование безопасных соединений
2. **Блокировка опасного контента** — защита от вредоносных и обманных сайтов
3. **Управление cookies** — настройка политики принятия cookies
4. **Разрешения сайтов** — контроль доступа к камере, микрофону, местоположению

## Системные требования

Для оптимальной работы Waterfox на Linux необходимо:

- **Процессор:** Intel Core 2 или AMD FX или выше
- **Оперативная память:** минимум 2 ГБ (рекомендуется 4 ГБ и более)
- **Место на диске:** не менее 500 МБ свободного пространства
- **Система:** 64-битная версия Linux
- **Библиотеки:** GTK+ 3.14 или выше, GLib 2.42 или выше

## Версии Waterfox

Доступны две основные версии браузера:

**Waterfox Current (G)** — современная версия с актуальными веб-технологиями и улучшенной производительностью. Рекомендуется для повседневного использования.

**Waterfox Classic** — версия с поддержкой устаревших расширений и классического интерфейса Firefox. Подходит для пользователей, нуждающихся в legacy-плагинах.
