# Vivaldi

Vivaldi — мощный интернет-браузер, созданный компанией Vivaldi Technologies на основе движка Blink. Разработан создателем Opera (Йоном Стефенсоном фон Тецнером) специально для опытных пользователей, ценящих гибкость, настройку и контроль над своим браузером. Первая версия вышла в 2016 году и включает множество функций классической Opera 12, таких как боковая панель, визуальные закладки, заметки, группировка вкладок и многое другое.

## Основные возможности

**Встроенные инструменты** — Vivaldi включает почтовый клиент, календарь, задачи, RSS-ленты и заметки прямо в браузере без необходимости установки расширений.

**Управление вкладками** — продвинутые функции работы с вкладками: группировка, размещение на двух панелях, мозаичное расположение для одновременного просмотра нескольких страниц.

**Персонализация** — полная настройка интерфейса, включая расположение панелей, цветовые схемы, темы, горячие клавиши и жесты мыши.

**Приватность** — встроенный блокировщик рекламы и трекеров, сквозное шифрование синхронизации, отсутствие сбора данных о пользователях.

**Proton VPN** — интегрированный VPN с активацией в один клик для защиты конфиденциальности.

**Цепочки команд** — возможность создавать наборы команд для автоматизации повторяющихся действий одним нажатием клавиши.

## Установка

Выберите подходящий способ установки Vivaldi для вашей системы:

::: code-group

```shell [UBUNTU, DEBIAN, MINT, POPOS]
curl -fSsL https://repo.vivaldi.com/archive/linux_signing_key.pub | sudo gpg --dearmor | sudo tee /usr/share/keyrings/vivaldi.gpg > /dev/null

echo "deb [arch=amd64 signed-by=/usr/share/keyrings/vivaldi.gpg] https://repo.vivaldi.com/archive/deb/ stable main" | sudo tee /etc/apt/sources.list.d/vivaldi.list

sudo apt update
sudo apt install vivaldi-stable
```

```shell [FEDORA]
sudo dnf config-manager --add-repo https://repo.vivaldi.com/archive/vivaldi-fedora.repo

sudo dnf install vivaldi-stable
```

```shell [OPENSUSE]
sudo zypper addrepo https://repo.vivaldi.com/archive/vivaldi-suse.repo

sudo zypper install vivaldi-stable
```

```shell [ARCH]
yay -S vivaldi
```

```shell [ALT LINUX, EPM PLAY]
epm play vivaldi
```

```shell [REDOS]
wget https://downloads.vivaldi.com/stable/vivaldi-stable-latest.x86_64.rpm

sudo dnf install ./vivaldi-stable-latest.x86_64.rpm
```

```shell [FLATPAK]
flatpak install flathub com.vivaldi.Vivaldi
```

```shell [SNAP]
sudo snap install vivaldi
```

```shell [EPM]
epm install vivaldi
```

:::

::: info
После установки через пакет DEB или RPM репозиторий Vivaldi автоматически добавляется в систему для получения обновлений.
:::

::: warning
При использовании Flatpak или Snap некоторые функции интеграции с системой могут работать ограниченно. Для полной функциональности рекомендуется использовать нативные пакеты.
:::

## Особенности работы

### Интеграция с рабочим окружением

Vivaldi автоматически адаптируется к настройкам вашей системы:

- Поддержка светлой и тёмной темы
- Масштабирование интерфейса
- Использование системных шрифтов
- Нативная поддержка Wayland

Для принудительного запуска в Wayland-режиме:
```sh
vivaldi --ozone-platform-hint=auto
```

### Управление вкладками

**Группировка вкладок** — объединяйте связанные вкладки в стеки для экономии места и удобной организации.

**Мозаичное расположение** — просматривайте до 4 вкладок одновременно в одном окне, разделив экран на части.

**Двухуровневые панели** — размещайте группы вкладок на двух панелях для работы с большим количеством открытых страниц.

**Гибернация вкладок** — автоматически выгружайте неактивные вкладки из памяти для повышения производительности.

### Встроенный почтовый клиент

Vivaldi Mail позволяет управлять электронной почтой прямо в браузере:

1. Откройте боковую панель и выберите «Почта»
2. Добавьте учётные записи (поддержка IMAP и POP3)
3. Используйте фильтры для организации писем
4. Интегрируйте календарь для управления событиями

::: tip
Почтовый клиент поддерживает несколько учётных записей и синхронизируется между устройствами с сквозным шифрованием.
:::

### Заметки и инструменты

**Менеджер заметок** — создавайте заметки с поддержкой форматирования, вложений и скриншотов прямо во время просмотра веб-страниц.

**Захват страниц** — делайте скриншоты всей страницы или выбранной области и сохраняйте их в заметках.

**Перевод страниц** — встроенный переводчик для быстрого перевода веб-страниц на другие языки.

## Полезные советы и дополнения

### Запуск Vivaldi в оконном интерфейсе Wayland

По умолчанию Vivaldi поддерживает запуск в Wayland. Для проверки текущего режима:

1. Введите в адресную строку `vivaldi://gpu`
2. Найдите строку «Ozone platform»
3. Если указан X11, измените настройку

Альтернативный способ через терминал:
```sh
vivaldi --enable-features=UseOzonePlatform --ozone-platform=wayland
```

::: info
Современные версии Vivaldi автоматически определяют окружение и запускаются в правильном режиме.
:::

### Настройка жестов мыши

Vivaldi поддерживает мощную систему жестов для управления браузером:

1. Откройте Настройки → Мышь
2. Включите «Жесты мышью»
3. Настройте существующие жесты или создайте собственные
4. Используйте правую кнопку мыши + движение для выполнения команд

### Создание цепочек команд

Автоматизируйте повторяющиеся действия с помощью цепочек команд:

1. Откройте Настройки → Быстрые команды → Цепочки команд
2. Нажмите «Добавить цепочку команд»
3. Выберите последовательность действий
4. Назначьте горячую клавишу или жест мыши
5. Запускайте цепочку одним нажатием

Примеры полезных цепочек:
- Открытие набора часто используемых сайтов
- Очистка данных браузера и перезагрузка страницы
- Создание заметки со скриншотом текущей страницы

### Использование рабочих пространств

Организуйте работу с помощью рабочих пространств:

1. Откройте боковую панель → Рабочие пространства
2. Создайте рабочие пространства для разных задач (Работа, Личное, Разработка)
3. Назначьте каждому рабочему пространству свои вкладки
4. Переключайтесь между пространствами горячими клавишами

::: tip
Используйте автоматические правила для автоматического распределения сайтов по рабочим пространствам.
:::

## Решение проблем

### Не воспроизводится видео в Vivaldi

Если возникают проблемы с воспроизведением видео:

**Проверка кодеков:**

::: code-group

```shell [UBUNTU, DEBIAN, MINT, POPOS]
sudo apt install ubuntu-restricted-extras
```

```shell [FEDORA]
sudo dnf install ffmpeg
```

```shell [ARCH]
sudo pacman -S ffmpeg
```

:::

**Проверка настроек:**
1. Введите `vivaldi://flags` в адресную строку
2. Найдите `#external-media`
3. Установите значение «Enabled»
4. Перезапустите браузер

### Проблемы с аппаратным ускорением

При графических артефактах или проблемах с производительностью:

1. Откройте `vivaldi://gpu`
2. Проверьте статус аппаратного ускорения
3. Если нужно, отключите через `vivaldi://flags`:
   - Найдите `#disable-accelerated-video-decode`
   - Установите «Enabled»
4. Для NVIDIA драйверов попробуйте:
   ```sh
   vivaldi --disable-gpu-compositing
   ```

### Сброс настроек браузера

При критических проблемах с настройками:

```sh
# Полный сброс настроек
rm -rfv ~/.config/vivaldi/

# Очистка только кэша
rm -rfv ~/.config/vivaldi/Default/Cache
rm -rfv ~/.config/vivaldi/Default/GPUCache
```

::: warning
Сброс настроек удалит все локальные данные. Убедитесь, что важные данные синхронизированы или сохранены.
:::

### Проблемы с синхронизацией

Если синхронизация не работает:

1. Проверьте подключение к интернету
2. Откройте Настройки → Синхронизация
3. Выйдите из аккаунта и войдите снова
4. Убедитесь, что выбраны правильные элементы для синхронизации
5. Проверьте, что сквозное шифрование настроено корректно

## Дополнительные настройки

### Оптимизация производительности

Для повышения скорости работы браузера:

1. Включите гибернацию фоновых вкладок:
   - Настройки → Вкладки → Гибернация фоновых вкладок
2. Ограничьте количество процессов:
   ```sh
   vivaldi --process-per-site
   ```
3. Отключите анимации в интерфейсе:
   - Настройки → Внешний вид → Использовать анимацию

### Блокировка рекламы и трекеров

Настройте встроенный блокировщик для максимальной защиты:

1. Откройте Настройки → Конфиденциальность и безопасность
2. Включите «Блокировать трекеры и рекламу»
3. Выберите уровень блокировки:
   - Список блокировки трекеров
   - Список блокировки рекламы
4. Добавьте собственные правила фильтрации при необходимости

### Настройка внешнего вида

Персонализируйте интерфейс под свои предпочтения:

1. **Темы** — создайте собственную цветовую схему или используйте готовые темы
2. **Расположение элементов** — перемещайте адресную строку, панель вкладок, боковую панель
3. **Масштабирование** — настройте размер интерфейса для комфортного просмотра
4. **Фон** — установите собственное изображение на стартовую страницу

Для автоматического переключения темы в соответствии с системой:
- Настройки → Темы → Использовать системную тему

### Работа с расширениями

Vivaldi совместим с расширениями из Chrome Web Store:

1. Перейдите на [chrome.google.com/webstore](https://chrome.google.com/webstore)
2. Найдите нужное расширение
3. Нажмите «Установить»
4. Управляйте расширениями через `vivaldi://extensions`

::: tip
Многие функции, доступные только через расширения в других браузерах, уже встроены в Vivaldi. Проверьте возможности браузера перед установкой дополнительных расширений.
:::

### Использование Proton VPN

Vivaldi включает интеграцию с Proton VPN:

1. Откройте боковую панель → VPN
2. Создайте аккаунт Proton или войдите в существующий
3. Выберите сервер и подключитесь
4. Следите за статусом подключения через индикатор

## Синхронизация между устройствами

Vivaldi Sync позволяет синхронизировать данные между компьютером и мобильными устройствами:

**Настройка синхронизации:**
1. Откройте Настройки → Синхронизация
2. Создайте аккаунт Vivaldi или войдите
3. Создайте пассфразу для сквозного шифрования
4. Выберите данные для синхронизации:
   - Закладки
   - Пароли
   - История
   - Заметки
   - Настройки
   - Вкладки

::: warning
Пассфраза для сквозного шифрования не восстанавливается! Сохраните её в надёжном месте, например в менеджере паролей.
:::

**Синхронизация открытых вкладок:**
- Доступ к вкладкам с других устройств через боковую панель
- Быстрое открытие страниц, просматриваемых на телефоне или планшете
